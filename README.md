# 📝 NoteScanner

A simple iOS app that scans handwritten notes and converts them into digital, text-based notes.

## 🚀 Overview

NoteScanner uses the device camera to capture handwritten notes and transform them into editable text inside the app.

## ✨ Features

* 📸 Capture notes using the camera
* 🔍 Detect document edges
* 🧠 Convert handwritten text into digital text
* 📝 View and manage scanned notes

## 🛠 Tech Stack

* Swift
* SwiftUI
* AVFoundation
* Vision (for text recognition)

## 📌 Goal

This project is built to explore camera handling, real-time processing, and text recognition on iOS.
